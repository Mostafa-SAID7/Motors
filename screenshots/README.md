# Motors - Application Screenshots

This folder contains screenshots and visual documentation of the Motors car showroom application.

## Screenshot Categories

### Dashboard
- `dashboard.png` - Main dashboard view with car listings

### Car Management
- `cars-list.png` - Cars list view with filters
- `car-detail.png` - Individual car detail page
- `car-form.png` - Car form for adding/editing

### Features
- `search.png` - Search functionality
- `filters.png` - Advanced filtering options
- `responsive-mobile.png` - Mobile responsive design

### UI Components
- `header.png` - Navigation header
- `footer.png` - Footer section
- `modals.png` - Dialog/modal examples

## Adding Screenshots

1. Take a screenshot of the feature
2. Save as PNG format
3. Name descriptively (e.g., `feature-name.png`)
4. Add to this folder
5. Update this README with description

## Screenshot Guidelines

- **Resolution**: 1920x1080 or higher
- **Format**: PNG (lossless)
- **Size**: Keep under 2MB per image
- **Content**: Show full feature/page
- **Naming**: Use kebab-case (e.g., `car-detail-page.png`)

## Usage

These screenshots are used for:
- GitHub README documentation
- Project presentations
- Feature demonstrations
- Bug reports (attach relevant screenshots)
- User guides
