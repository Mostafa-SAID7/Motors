# موتورز - تطبيق معرض السيارات

## 🚗 نظرة عامة

**موتورز** هو تطبيق ويب حديث لإدارة معرض السيارات مبني باستخدام Angular 19 و Tailwind CSS و PrimeNG. يوفر تجربة مستخدم احترافية لعرض وإدارة السيارات.

## ✨ المميزات الرئيسية

### 📊 لوحة التحكم
- إحصائيات عدد السيارات
- عدد السيارات الجديدة والمستعملة
- عرض السيارات المميزة

### 🚙 قائمة السيارات
- عرض جميع السيارات
- فلاتر متقدمة (الماركة، السعر، الحالة)
- بحث شامل
- ترتيب وتصفية

### 🔍 تفاصيل السيارة
- معرض صور
- مواصفات كاملة
- السعر والحالة
- أزرار الحجز والاتصال

### ➕ إضافة/تعديل السيارة
- نموذج كامل
- التحقق من الصحة
- رفع الصور
- حفظ البيانات

## 🛠️ المتطلبات

- Node.js 18+
- npm 10+
- Angular CLI 21+

## 📦 التثبيت

```bash
# استنساخ المشروع
cd Motors

# تثبيت المكتبات
npm install --legacy-peer-deps

# تشغيل خادم التطوير
npm start
```

## 🚀 البدء السريع

### 1. تشغيل الخادم
```bash
npm start
```

### 2. فتح المتصفح
```
http://localhost:4200
```

### 3. استكشاف التطبيق
- الصفحة الرئيسية: `/`
- قائمة السيارات: `/cars`
- تفاصيل السيارة: `/cars/:id`
- إضافة سيارة: `/cars/add`
- تعديل سيارة: `/cars/:id/edit`

## 🔄 إدارة الحالة

استخدام **Angular Signals** (مدمج في Angular):

```typescript
// تعريف Signal
private cars = signal<Car[]>([]);

// الوصول للقراءة فقط
getCarsSignal() {
  return this.cars.asReadonly();
}

// التحديث
this.cars.set(updatedCars);
```
